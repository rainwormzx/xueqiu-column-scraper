# 雪球专栏文章爬虫使用指南

## 准备工作

1. 安装依赖：
```bash
pip install selenium
```

2. 确保ChromeDriver已安装：
   - 项目目录中已有 `chromedriver.exe`
   - 或者确保ChromeDriver在系统PATH中

## 使用方法

1. 运行脚本：
```bash
python xueqiu_column_scraper.py
```

2. 脚本启动后会打开浏览器，显示雪球网首页

3. **重要**：手动扫码登录雪球账号

4. 登录完成后，在控制台按回车键继续

5. 脚本将自动导航到指定专栏并开始抓取文章

6. 抓取的数据将保存到 `xueqiu_column_articles.csv` 文件中

## 配置

可在脚本开头修改以下配置：

```python
# 目标专栏URL
COLUMN_URL = 'https://xueqiu.com/8790885129/column'

# CSV输出文件名
CSV_FILENAME = 'xueqiu_column_articles.csv'

# 开始页码
START_PAGE = 1
```

## 注意事项

- 请遵守雪球网的使用条款
- 登录会话可能有过期时间，请适时重新登录
- 部分内容可能需要付费或权限才能访问
- 合理使用，避免过于频繁的请求
