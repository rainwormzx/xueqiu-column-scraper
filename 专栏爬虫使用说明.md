# 雪球专栏文章爬虫使用说明

## 文件说明
- `xueqiu_column_scraper.py`: 专门用于抓取雪球专栏文章的爬虫脚本

## 功能特点
此脚本基于原有的`xueqiu_scraper.py`进行了优化，专门用于抓取雪球专栏文章内容，主要特点包括：

1. **专门针对专栏页面设计**：识别专栏文章列表和详情页的特定DOM结构
2. **完整的文章内容抓取**：不仅抓取摘要，还会访问每篇文章详情页获取完整内容
3. **保留原有功能**：复用登录、反反爬虫、数据保存等通用组件
4. **数据字段更丰富**：包含标题、时间、内容、统计数据等

## 配置说明
在脚本开头可以配置以下参数：

```python
# 目标专栏URL (比如 https://xueqiu.com/8790885129/column )
COLUMN_URL = 'https://xueqiu.com/8790885129/column'  # 请替换为实际的专栏URL
CSV_FILENAME = 'xueqiu_column_articles.csv'  # 输出文件名

# 从哪个页开始抓取（默认为 1）
START_PAGE = 1
```

## 使用方法

1. 确保已安装必要的依赖包：
   ```
   pip install selenium
   ```

2. 下载并配置ChromeDriver（版本需与你的Chrome浏览器匹配）

3. 运行脚本：
   ```
   python xueqiu_column_scraper.py
   ```

4. 根据提示在浏览器中扫码登录雪球账号

5. 程序将自动访问指定的专栏页面并开始抓取文章

## 主要改进

1. **页面结构适配**：
   - 使用了适合专栏页面的CSS选择器
   - 识别专栏特有的文章列表结构

2. **深度内容提取**：
   - 对每篇文章链接进行单独访问
   - 提取完整的文章标题和正文内容

3. **增强的解析能力**：
   - 更多的标题和内容选择器选项
   - 智能处理不同的页面布局

4. **数据结构优化**：
   - CSV输出包含更丰富的文章信息
   - 包括文章ID、标题、发布时间、内容等

## 注意事项

1. 请遵守雪球网的使用条款和robots.txt协议
2. 合理控制请求频率，避免对服务器造成过大压力
3. 如遇到反爬虫措施，可根据需要调整延迟时间
4. 登录后的会话可能有时效性，长时间运行可能需要重新登录

## 常见问题

- **无法找到文章元素**：可能专栏页面结构发生变化，需要更新CSS选择器
- **登录失败**：检查网络连接，确认能正常访问雪球网站
- **内容不完整**：某些文章可能是付费内容，普通账号无法访问全部内容
